# 🚦 JavaFX Traffic Simulation System
A real-time graphical traffic simulation built using **Java 21** and **JavaFX**.

## 🚀 Features
* **4-Way Intersection:** Synchronized traffic light logic (North-South vs East-West).
* **Object-Oriented Design:** Custom classes for `car` and `TrafficLights`.
* **Sprite Animation:** Uses `ImageView` for realistic vehicle and signal visuals.
* **Collision Avoidance:** Cars detect Red lights and stop automatically.

## 🛠️ Tech Stack
* **Language:** Java 21
* **Framework:** JavaFX 21
* **Build Tool:** Maven
* **Version Control:** Git & GitHub
